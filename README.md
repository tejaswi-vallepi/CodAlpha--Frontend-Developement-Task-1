# Responsive Image Gallery

## Overview

This project is a **Responsive Image Gallery** developed using **HTML, CSS, and JavaScript**. It allows users to browse images in an attractive gallery layout, filter images by category, and view images in a fullscreen lightbox with navigation controls.

The application demonstrates core front-end development concepts such as responsive design, DOM manipulation, event handling, image filtering, and interactive user interfaces.

## Features

### 1. Responsive Grid Layout

* Automatically adjusts to different screen sizes.
* Uses CSS Grid for a clean and flexible layout.
* Provides an optimal viewing experience on desktops, tablets, and mobile devices.

### 2. Category-Based Filtering

* Users can filter images by:

  * All
  * Nature
  * Animals
  * City
* JavaScript dynamically shows or hides images based on the selected category.

### 3. Interactive Hover Effects

* Smooth image zoom effect on hover.
* Card elevation effect using CSS transforms and shadows.
* Enhances visual appeal and user engagement.

### 4. Lightbox Image Viewer

* Displays selected images in fullscreen mode.
* Dark overlay focuses attention on the selected image.
* Easy image viewing experience without leaving the page.

### 5. Image Navigation

* Previous and Next buttons allow users to browse images inside the lightbox.
* Circular navigation ensures continuous image browsing.

### 6. Keyboard Controls

* Right Arrow → Next Image
* Left Arrow → Previous Image
* Escape Key → Close Lightbox

### 7. Mobile-Friendly Design

* Responsive layout adapts to smaller screens.
* Lightbox images resize appropriately for mobile devices.

## How It Works

1. Images are displayed in a responsive grid.
2. Users select a category to filter images.
3. Clicking an image opens it in a fullscreen lightbox.
4. Navigation buttons and keyboard shortcuts allow image browsing.
5. The lightbox can be closed using the close button or the Escape key.
